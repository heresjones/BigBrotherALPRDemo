# Vehicle photo credits

All photos in this folder are sourced from Wikimedia Commons under free
licenses. Each has been resized and had any real, legible license plate
**permanently pixelated in the image file itself** (not just hidden with
CSS) before being added to this repo — this app's whole design principle
is that no real plate ever appears anywhere in it, including in the raw
asset files. The fictional plate text shown in the app is rendered on top
by the `VehiclePhoto` component at display time; it is not baked into
these files.
