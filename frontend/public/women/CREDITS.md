# Women — face photo credits

All photos in this folder are AI-generated (synthetic) faces from
[Generated.Photos](https://generated.photos/) — nobody in this folder is a
real person. That sidesteps the "no real people" concern this project has
followed elsewhere (see `../vehicle-photos/CREDITS.md`), but introduces a
different one: these were fetched from Generated.Photos' public API preview
tier, and the URLs carry a `wm:1` (watermark) parameter, meaning they're
preview/browsing assets, not necessarily licensed for use in a shipped app.

**Before these appear in a public build or a screenshot that leaves this
machine, confirm licensing with Generated.Photos.** Fine for local dev/demo
use in the meantime.

Files are named by their Generated.Photos image `id`.
